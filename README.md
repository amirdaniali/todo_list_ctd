## Project Overview

This project is a fully interactive Todo List built with React.
It demonstrates core React concepts including:

- Component-based architecture
- State management with `useState`, `useReducer`, and context
- Controlled form inputs
- Sorting and filtering logic
- Inline component styling
- Real-time UI updates
- Basic validation and error handling

The app includes:

- Add Todo form with validation
- Filter input for searching todos
- Sort controls (by title or creation date, ascending/descending)
- Todo list with update, complete, and edit functionality
- Unified styling system across all components
- React Router based navigation across the pages

This project evolves week by week as part of the course, with Week 1 focusing on setup and basic functionality.

## Installation & Setup

After cloning the repository:

```bash
npm i
npm run dev
```

Then open the local development URL shown in your terminal.

## Goal of the Project

The purpose of this project is to practice building a real-world React application from scratch while learning:

- Component composition
- State and event handling
- Form management
- UI styling patterns
- Clean code structure
- Debugging and iterative development
